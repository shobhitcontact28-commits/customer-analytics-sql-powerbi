# Power BI Dashboard

This folder contains the interactive Power BI dashboard developed for the customer analytics project.

The dashboard visualizes key business insights derived from the cleaned dataset and SQL analysis.

## Dashboard Features

The dashboard includes the following visualizations:

### Executive KPIs
- Total Customers
- Total Revenue
- Average Customer Lifetime Value (CLV)
- Average Purchases per Customer

### Customer Segmentation
- Revenue by Income Segment
- Revenue by Product Category

### Customer Behavior Analysis
- Customer Lifetime Value by Membership Status
- Customer Lifetime Value by Satisfaction Level

### Geographic Insights
- Top Revenue Generating Countries

### Interactive Filters
The dashboard includes slicers to filter results by:

- Country
- Income Level
- Favorite Category

These filters allow users to dynamically explore the dataset and analyze revenue patterns across different customer segments.

## Tools Used

- Power BI Desktop
- SQL Server (SSMS)
